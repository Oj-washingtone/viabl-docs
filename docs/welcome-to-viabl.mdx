Viabl is a documentation platform that lets you write in MDX, configure with a single JSON file, and deploy anywhere. It gives your docs the same care you give your product.

## What is Viabl?

Viabl is made up of three parts that work together:

<CardGroup cols={3}>
  <Card title="CLI" icon="Terminal">
    The `viabl` command line tool. Use it to create projects, preview locally,
    and build for production.
  </Card>
  <Card title="Renderer" icon="Layout">
    The Next.js app that turns your MDX files into a beautiful, fast
    documentation site.
  </Card>
  <Card title="Content server" icon="Server">
    A lightweight Express server that reads your docs from disk and serves them
    to the renderer.
  </Card>
</CardGroup>

## Get started in minutes

<Steps>
  <Step title="Install the CLI">
    Install the Viabl CLI globally with npm.

```bash
    npm install -g @viablkit/cli
```

  </Step>
  <Step title="Create your project">
    Scaffold a new docs project from the starter template.

```bash
    viabl init my-docs
    cd my-docs
```

  </Step>
  <Step title="Start writing">
    Start the local dev server and open your browser.

```bash
    viabl dev
```

    Your docs are live at `http://localhost:7777`. Edit any MDX file and the browser reloads automatically.

  </Step>
</Steps>

## Core concepts

<AccordionGroup>
  <Accordion title="docs.json — your single config file" defaultOpen>
    Everything about your docs lives in one file — navigation structure, branding, colors, fonts, versioning, and more. No scattered config files, no complex setup.
```json
    {
      "$schema": "https://viabl.dev/schema/docs.json",
      "name": "My Docs",
      "colors": {
        "primary": "#ff5800"
      },
      "navigation": [
        {
          "group": "Getting Started",
          "pages": ["introduction", "getting-started"]
        }
      ]
    }
```
  </Accordion>
  <Accordion title="MDX — write with superpowers">
    Your pages are MDX files — Markdown with the ability to use React components inline. Write prose, embed code blocks, add callouts, cards, tabs, and more without leaving your editor.
  </Accordion>
  <Accordion title="Self-hosted — deploy anywhere">
    Viabl runs on your own infrastructure. Build a production bundle with `viabl build` and deploy it to any VPS, Docker container, or hosting platform that runs Node.js.
  </Accordion>
</AccordionGroup>
