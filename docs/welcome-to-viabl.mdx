Viabl is a documentation platform that lets you write in MDX, configure with a single JSON file, and deploy anywhere. It gives your docs the same care you give your product.

## What is Viabl?

Viabl is made up of three parts that work together:

<CardGroup cols={3}>
  <Card title="CLI" icon="Terminal">
    The `viabl` command line tool. Use it to create projects, preview locally,
    and build for production.
  </Card>
  <Card title="Renderer" icon="Layout">
    The Next.js app that turns your MDX files into a beautiful, fast
    documentation site.
  </Card>
  <Card title="Content server" icon="Server">
    A lightweight Express server that reads your docs from disk and serves them
    to the renderer.
  </Card>
</CardGroup>
